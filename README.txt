Тестовое задание (выполнить на JAVA)
 
Необходимо создать приложение состоящее из 1 - го компонента (Activity или Fragment ).

    Реализовать данное задание с применением RxJava 2, Dagger  2 , MVP.
Плюсом будет использование Room и  Moxy.

Реализовать:
● 	база данных (5 полей) и заполнить любыми данными
● 	вывод в (Activity или Fragment ) в виде списка
●    live search по БД с отображением в (Activity или Fragment ) (Toolbar или Custom solution)
● 	реализовать сортировку списка (любой параметр) по клику
